Given a causal model, not only are we able to determine the causal relationships among variables of interest, but also to estimate the effects quantitatively from observational data. Even when the model is partially identified, there are robust strategies to deal with unobserved biases. The construction of causal models often relies heavily on human expertise, which obviously fails to account for all possible scenarios. This article focuses on research progress in discovering causal structures from data. Additionally, I attempt to introduce some advanced topics related to causal inference for spatio-temporal data.

Causal Discovery from Observational Data

Causal learning algorithms from observational data can be broadly categorized into Constraint-based (CB) and Score-based (SB) approaches. The task, again, is to find the causal graph that aligns with the true data-generating process. CB methods uncover a set of causal graphs up to the Markov equivalence class, through a series of conditional independence tests. The tests are often implemented independently with binary decisions of either accepting or rejecting the null hypothesis. SB is a more efficient method that aims to evaluate the quality of a candidate causal graph according to a score function i.e., graphs with higher scores are better fits. Thus, the function must satisfy certain properties to be considered an appropriate goodness-of-fit test.

This discussion is not intended to be a survey of causal discovery algorithms, rather I will only delve into these particular works

  1. Peter-Clark Algorithm: the foundational CB method, ready to use in pcalg
  2. Generalized Score Functions: a unified SB method
  3. Greedy Equivalence Search : a widely used search algorithm, ready to use in pcalg
  4. DAGs with NO TEARS: a optimization-based SB approach, ready to use in CausalNex
